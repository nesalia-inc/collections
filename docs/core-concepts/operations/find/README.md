# Find Operations

Learn how to query records from collections.

## Operations

- [find](./find.md) - Get all records from a collection (paginated)
- [findById](./find-by-id.md) - Get a single record by ID
- [findFirst](./find-first.md) - Get the first record matching conditions
- [pagination](./pagination.md) - Pagination strategies (offset and cursor)
- [count](./count.md) - Count records matching conditions
- [exists](./exists.md) - Check if a record exists
